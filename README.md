# GMVIS
Greater Manchester Vis

## Website overview
This repository contains a static multi-page community website for Greater Manchester Vis Community Group, including:
- Group overview and benefits for new users
- Sports and activities page
- Blog page with in-browser add/update support (localStorage)
- General enquiry/contact form
- Complaints form

## Run locally
From the repository root, run a simple static server such as:

```bash
python3 -m http.server 8080
```

Then open `http://localhost:8080/index.html`.
